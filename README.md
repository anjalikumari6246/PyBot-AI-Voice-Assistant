# 🤖 PyBot – Generative AI Voice Assistant

PyBot is an AI-powered desktop voice assistant built using Python, Tkinter, and the Google Gemini API. It supports both voice and text interactions through a simple graphical interface.

## Features

* Real-time AI conversations using Gemini API
* Voice-to-text input using SpeechRecognition
* Text-to-speech responses
* Dark-themed Tkinter GUI
* Conversation history storage
* Multithreaded architecture for responsive interactions

## Technologies Used

* Python
* Tkinter
* Google Gemini API
* SpeechRecognition
* SoundDevice
* pyttsx3
* NumPy
* SciPy

## Installation

```bash
pip install google-genai SpeechRecognition pyttsx3 sounddevice scipy numpy
```

## Run

```bash
python pybot1.py
```
